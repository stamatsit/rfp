Executive Summary

The page works. Nothing else does.

Coe's Majors page is the primary gateway for prospective students. Despite 82 program listings, it provides almost zero information about any of them.

🔍

No Way to Search or Filter

82 programs in one flat list. No keyword search, no category filters, no department grouping. Students who don't know what they want have zero way to explore.

📄

Zero Program Descriptions

Every major is a name and a link. Not one sentence about what students will study, what careers it leads to, or why Coe's program stands out.

♿

Accessibility Failures

Zero alt text. No ARIA labels. Inaccessible symbols. Missing semantic HTML. Fails WCAG 2.1 Level A — the bare minimum — with potential ADA legal exposure.

📈

Invisible to Search & Social

No meta description, OG tags, Twitter Cards, Schema.org, or canonical URL. Social previews are blank. Google gets zero structured information.

🔒

Privacy Violations

5 tracking scripts fire before consent. Hotjar records session replays. No cookie banner. Likely violates GDPR, CCPA, and emerging state privacy laws.

🏆

No Trust Signals

No rankings, accreditation badges, graduation rates, employment data, class sizes, or student-faculty ratio. Zero proof of value from a quality institution.

Accessibility

WCAG 2.2 / ADA compliance

8/100
9 issues
Critical
No alt text on any image
All 8 images have zero alt text. Violates WCAG 2.1 Level A (1.1.1). ADA Title III liability.
Critical
No ARIA labels on navigation
Dropdowns lack aria-expanded, aria-haspopup. Keyboard users can't determine menu state.
Critical
Search field has no label element
Placeholder text only, not a proper <label>. Violates WCAG 2.1 Level A (1.3.1).
Critical
Symbol legend is inaccessible
Screen readers say "caret" not "also a minor." Legend buried at page bottom.
High
No visible focus indicators
Keyboard users can't see focus. Violates WCAG 2.1 AA (2.4.7).
High
Missing semantic landmarks
No <nav>, <main>, <footer> elements. Screen readers can't jump between regions.
High
No lang attribute
Screen readers may use wrong pronunciation. Violates WCAG 2.1 A (3.1.1).
Medium
No skip-to-section links
80+ items need skip links to Minors and Pre-Professional sections.
Medium
No high-contrast mode
Many higher-ed sites now offer built-in accessibility toolbars.
