The Formula

Every category starts at 100.

Issues subtract from there. One simple rule, applied to all 11 categories.

A Real Example

From issues to a final score.

One scan, all the way through.

Step 01
Score each category
The scanner found a few image problems and one missing landmark. Each category got a score using the formula on the previous page.
images    = 65  // 2 errors, 1 warning
contrast  = 100
forms     = 100
landmarks = 85  // 1 error
headings  = 95
structure = 100
security  = 90
Step 02
Collapse into buckets
The seven categories average together into three buckets — Accessibility, Structure, Security.
Accessibility = (65 + 100 + 100 + 85) ÷ 4 = 87.5
Structure     = (95 + 100) ÷ 2 = 97.5
Security      = 90
Step 03
Apply the weights
Each bucket gets multiplied by its weight, then everything adds up.
87.5  × 55%  = 48.125
97.5  × 30%  = 29.250
90.0  × 15%  = 13.500
───────────────── Total        = 91
91
Final score: 91 / 100
02 · The Headings Check

One rule. One answer.

The simpler of the two tools. No score — just compliant or not compliant.

currentLevel − previousLevel ≤ 1
Going deeper Levels can only increase by one
Going shallower Unrestricted (h4 → h1 is fine)
First heading Always allowed, any level
Required At least one h1 on the page

This is the exact heading-order rule from axe-core, the engine used by Lighthouse and most accessibility auditors.

Remember This

Two things worth knowing.

One Honest Limit
Color contrast isn't checked yet.
Color contrast — the most common accessibility issue on the web — requires a real browser to compute rendered styles. Our scanner runs without a browser for speed, so contrast scoring is skipped. Treat the score as "everything except contrast." A future update will add a separate contrast pass.
Rule of Thumb
Errors hurt 3× more than warnings.
Fixing one error moves the score the same as fixing three warnings. When you're triaging, sort by severity first.
Highest Leverage
Accessibility is 55% of the overall.
It's the heaviest bucket by far. If you can only fix one thing on a site, fixing accessibility moves the needle fastest.
